## Atletas do Brasil

Uma plataforma para pesquisar e conhecer mais sobre os atletas brasileiros.

### Como usar
1. Digite o nome do atleta ou esporte no campo de pesquisa.
2. Clique no botão "Pesquisar".
3. Os resultados serão exibidos abaixo.
